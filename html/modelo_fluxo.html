Como funciona o fluxo de situações?
O fluxo de situações é uma nova abordagem para a melhora das estatísticas no 2º Grau. Esta abordagem foca na situação do processo e nos eventos que podem ocorrer sobre os processos nestas situações. Os eventos previstos alteram a situação dos processos para uma nova situação.
Como é realizada a implementação do fluxo?
Por meio do grafo de situações é possível verificar quais as ações são esperadas dada a situação dos processos
  1. Notem que são desenhadas apenas as ações consideradas consistentes.
  2. Na implementação de fato, são incluídas algumas ações inconsistentes para que se melhore a apuração das estatísticas.
